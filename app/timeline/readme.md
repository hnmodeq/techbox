# خط‌زمان تکنولوژی / Technology Timeline

## Route: `/timeline`

Public interactive timeline visualization for exploring technology history.

### Features
- 🔍 Zoom in/out (0.5x to 3x magnification)
- 🖱️ Pan left/right by dragging
- 📅 Dates in Solar Hijri (Persian calendar)
- 🎨 Card size based on event importance
- 📍 Temporal spacing (100px per year)
- 💬 Comments & likes (structure ready)
- 🌙 Dark theme optimized
- 🇮🇷 Persian/RTL ready

### Technology Stack
- Next.js 16 (React 19)
- Tailwind CSS 4
- Prisma ORM
- TypeScript
- Lucide React (icons)

### How to Use
1. Visit `/timeline` in your browser
2. Use zoom controls (bottom-right) to scale
3. Drag to pan across the timeline
4. Hover over cards for details
5. Click like/comment buttons (future feature)

### Admin Panel
Manage timeline events at `/admin/timeline`
- Create new events
- Edit existing events
- Delete events
- View all events chronologically

For detailed usage, see `readme/TIMELINE_USAGE.md`
